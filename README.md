# 客户聚类分析工具

## 项目简介
这是一个基于 K-Means 算法的客户聚类分析工具，用于对客户进行精准分群，帮助企业制定差异化的营销策略。

## 功能特点
- 📊 **自动数据处理**：支持 CSV 文件导入，自动处理缺失值
- 🔍 **智能聚类**：使用肘部法则和轮廓系数自动确定最优聚类数
- 📈 **可视化分析**：生成多种图表展示聚类结果
- 💡 **业务建议**：为每个客户群自动生成营销策略建议

## 核心技术

### K-Means 聚类算法
K-Means 是一种无监督学习算法，通过迭代将客户划分为 K 个群组，使得同组内客户特征相似，不同组间差异明显。算法步骤：
1. 随机选择 K 个初始聚类中心
2. 将每个客户分配到最近的聚类中心
3. 重新计算每个簇的中心点
4. 重复步骤 2-3 直到收敛

### 最优 K 值选择
- **肘部法则**：观察惯性值随 K 增加的下降趋势，在拐点处选择 K
- **轮廓系数**：衡量聚类质量，值越接近 1 表示聚类效果越好

## 使用方法

### 1. 安装依赖
```bash
pip install pandas numpy scikit-learn matplotlib seaborn
```

### 2. 准备数据
CSV 文件应包含以下列：
- 客户 ID
- 总消费金额
- 订单数量
- 平均订单价值
- 电子产品消费金额

### 3. 修改文件路径
在代码第 38 行修改为你的 CSV 文件路径：
```python
csv_file_path = "your_data.csv"
```

### 4. 运行分析
```bash
python customer_clustering.py
```

## 输出结果

- `cluster_distributions.png` - 各簇特征分布箱线图
- `cluster_comparison.png` - 客户群对比图
- `customer_clustering_results.csv` - 包含聚类标签的完整数据

## 应用场景
- 精准营销：针对不同客户群设计差异化营销方案
- 客户价值管理：识别高价值客户进行重点维护
- 流失预警：发现低活跃度客户并制定唤醒策略
- 产品推荐：根据客户偏好推送合适的产品
